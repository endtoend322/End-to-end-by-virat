🔐 Encryption Details:
- Algorithm: Fernet (AES-128 in CBC mode with PKCS7 padding)
- Key size: 32 bytes (256 bits)
- IV size: 16 bytes
- Authentication: HMAC with SHA256

📊 Database Tables:
- users - User information and keys
- messages - Encrypted message storage
- conversations - Conversation metadata
- cookies - Encrypted session cookies
- automation_logs - Automation action logs
- attachments - Encrypted file attachments

🚀 Quick Start:
1. Install dependencies: pip install -r requirements.txt
2. Run application: streamlit run streamlit_app.py
3. Access at: http://localhost:8501
