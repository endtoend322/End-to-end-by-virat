# 🔐 End-to-End Encryption Chat Application

## 🚀 Overview

A comprehensive, artistic, and fully functional end-to-end encryption chat application built with Streamlit. This application provides secure communication with advanced features for WhatsApp and Facebook platforms.

![E2E Encryption Chat](https://via.placeholder.com/800x400?text=E2E+Encryption+Chat+Application)

## ✨ Features

### 🔐 Security Features
- **End-to-End Encryption**: All messages are encrypted using Fernet (AES-128)
- **Secure Key Management**: Automatic key generation and secure storage
- **Encrypted Storage**: All data is encrypted at rest
- **Session Management**: Secure cookie and session handling

### 💬 Chat Features
- **Multi-Platform Support**: WhatsApp and Facebook integration
- **Real-Time Messaging**: Instant message delivery
- **Group Chat Support**: Multi-participant conversations
- **File Attachments**: Secure file sharing with encryption

### 🎨 Creative Interface
- **Artistic Design**: Beautiful white and light blue color scheme
- **Interactive UI**: Smooth animations and transitions
- **Responsive Design**: Works on all screen sizes
- **Live Console**: Real-time automation output monitoring

### ⚙️ Configuration Options
- **Target Settings**: Chat/E2E ID configuration
- **Name Prefix**: Custom name prefix management
- **Delay Settings**: Configurable time delays for automation
- **Cookie Management**: Single and multiple cookie support
- **File Upload**: Drag and drop file support
