# Automation Dashboard

Advanced Streamlit automation dashboard template.

## Features
- Background worker thread
- SQLite logging system
- File upload support
- Live logs viewer
- 24/7 deployment ready
- Clean production structure

## Installation

1. Create virtual environment
2. Install dependencies:

pip install -r requirements.txt

3. Run application:

streamlit run streamlite_app.py

## 24/7 Deployment

Deploy on:
- AWS EC2
- DigitalOcean VPS
- Railway
- Render
- Docker container
